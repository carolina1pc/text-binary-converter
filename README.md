# Text & Binary Converter

A simple and intuitive React application that allows users to convert text to binary and binary back to text in real time. Designed with clean UI, smooth interactions, and a lightweight structure for fast performance.

## Installation & Setup

Follow these steps to run the project locally:

1. Clone the repository
`git clone https://github.com/carolina1pc/text-binary-converter.git`

2. Navigate into the project folder
`cd text-binary-converter`

3. Install dependencies
`npm install`

4. Start the development server
`npm start`
